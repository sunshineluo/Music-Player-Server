# 简约播放器
![](./demo.png)
准备好内嵌歌曲图片的歌曲和对应的歌词文件，放在music文件夹内。然后启动项目运行即可使用。

### 使用方法

```base
pnpm install
npm run start
```
或者直接点击.bat脚本一键启动即可，前提是电脑需要有node环境！

项目启动后直接在端口9000启动，添加歌曲直接往music文件夹内添加歌曲即可。

### 键盘快捷键
- 上一首：`left`
- 下一首：`right`
- 播放/暂停：`space`
- 音量+：`up`
- 音量-：`down`
- 播放模式切换：`Ctrl`
- 播放列表：`L`
